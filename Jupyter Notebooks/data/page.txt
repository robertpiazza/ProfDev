class="listview__content"><div class="exercise--assignment exercise--typography"><h1 class="exercise--title">Defining constants with convenience functions</h1><div class=""><p>A constant is the simplest category of tensor. It can't be trained, which makes it a bad choice for a model's parameters, but a good choice for input data. Input data may be transformed after it is defined or loaded, but is typically not modified by the training process.</p>
<p>In this exercise, we will practice defining constants using some of the operations discussed in the video. Note that we have not imported the entire <code>tensorflow</code> API and will not import it for most exercises. You can complete this exercise using the operations <code>fill()</code>, <code>ones_like()</code>, and <code>constant()</code>, which have been imported from <code>tensorflow</code> version 2.0 and are available in the IPython shell.</p></div></div></div></div><div class="listview__section" data-onboarding="instructions" style="min-height: calc(100% - 33px);"><div><div role="button" class="listview__header"><div class="exercise--sidebar-header"><h5 class="dc-panel__title"><svg width="12" height="12" aria-label="checkmark_circle icon" class="dc-icon-checkmark_circle dc-u-color-grey-dark dc-u-mr-8" role="Img" fill="currentColor"><use xlink:href="/static/media/symbols.cace91b1.svg#checkmark_circle"></use></svg>Instructions</h5><span class="tag tag--xp">100 XP</span></div></div></div><div class="listview__content"><div><div class=""><div data-onboarding="instructions" class="exercise--instructions exercise--typography"><div class="exercise--instructions__content"><ul>
<li>Define a 3x4 tensor, <code>x</code>, using <code>fill()</code> where each element has the value 9.</li>
<li>Create a tensor, <code>y</code>, with the same shape as <code>x</code>, but with all values set to one using <code>ones_like()</code>.</li>
<li>Define a 1-dimensional <code>constant()</code>, <code>z</code>, which contains the following elements: <code>[1, 2, 3, 4]</code>.</li>
<li>Print <code>z</code> as a <code>numpy</code> array using <code>.numpy()</code>.</li>
